# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step1 : First import numpy module with a nickname as np
### Step 2: Store the matrix data row-wise in a variable(i.e; a)
### Step 3: Then find the inverse of that matrix using variable name (a) using the syntax " np.linalg.inv(a) "
### Step 4: Store the inverse value in other variable and print the result

## Program:
```
#Program to find the inverse of a matrix.
#Developed by: Aashif Ahamed S
#RegisterNumber: 212225040004
import numpy as np
a=np.array([[1,0,3],[-1,2,-2],[2,3,-1]])
i=np.linalg.inv(a)
print(i)
```
